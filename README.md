# Verbex
Verbex is a statically typed programming language which is more verbose than python, handles errors like rust, auto manages memory and is compiled with the speed of c and c++.

## Features
Verbex comes with many features. These features make this language unique and its verbosity makes it easier to learn.

### Compiled
The language is compiled. The entire language is made with highly optimized llvm. Due to this optimizatins it is very efficient when it comes to execution.

### Power of C
This langauge is made entirely by taking reference from C. Due to this you will see most of the features which you see in C here. This makes it easier for a person who knows C to graspp it.   
Also this language is made entirely for beginners as its verbosity allows the user to read it just like pseudocode.

### No Pointers
The language is inspired from C but it does not use pointers. The memory is handled automatically by the garbage collector.

### Highly Verbose Errors
The errors in verbex are highly verbose just as its syntax allowing developers to debug easily. The errors are designed in such wayy that an agent could easily pick it up.

## Tokens or Keywords
Verbex does'nt rely much on symbols. It only uses it for certain operations and everything else is left to the english words. One might think that more english words means more writing but this language is agentic first meaning code will look entirely like pseudocode making an agent to read and debug it easily.
